# 翁法罗斯（Amphoreus）与昔涟（Cyrene）调研记录 v1

## 文档目的

这份文档用于为 `Cybersoul` 项目后续的世界实例设计、陪伴体 persona 设计、场景与 NPC 建模提供调研基础。

当前版本聚焦：

- 翁法罗斯的世界本质与总体结构
- 主要地点与其叙事定位
- 昔涟（Cyrene）的核心人物设定
- 昔涟与关键角色的关系
- 对 `Cybersoul` 适配最有价值的原型启发

说明：

- 本文优先采用官方或半官方来源。
- 由于部分官方内容难以直接抓取，本文会使用 `Fandom` 上对 `HoYoLAB` 官方文章的转录页作为半官方辅助来源。
- 文中会尽量区分“明确事实”和“推断/适配判断”。
- 当前仍是 `v1`，并不等于翁法罗斯世界的完整百科，后续还需要继续补完六大城邦细节、更多 NPC、支线事件与任务顺序。

## 一、世界本质

### 1.1 明确事实

翁法罗斯并不是一颗普通意义上的自然星球。

根据公开资料，翁法罗斯在更深层真相上，是由 `Lygus` 借助 `Scepter δ-me13` 运行的实验世界。这个世界被放置在受 `Stellaron` 污染与 `memoria` 加密环境影响的隔离条件中，用于推动 `Irontomb` 的演化与相关实验流程。

在这个实验世界中：

- 世界以 `memoria + data` 的方式被构造与维持
- 泰坦、黄金裔等神话性存在，在底层机制上与“因子”和“电信号”有关
- 黑潮在实验层面可被理解为 `Destruction Equation` 的表现
- 翁法罗斯并非只经历一条历史，而是经历了极多轮循环与推演

这意味着翁法罗斯虽然在故事表层看起来是神话世界，但在底层更接近一个被运行、被观察、被记忆维持的沙盒世界。

### 1.2 对项目的价值

这一点与 `Cybersoul` 的赛博世界设定高度同构。

可复用的不是单纯的地名或角色名，而是这种世界逻辑：

- 世界不是“假”的，而是以另一种媒介存在
- 记忆、数据、信号与循环共同维持世界连续性
- 人格与历史不是单轮对话，而是长程积累
- 场景、事件、关系都可以被设计为“正在发生的生活”

## 二、世界的表层神话结构

### 2.1 神话外壳

在世界内叙事中，翁法罗斯被理解为一个由泰坦创造天地、秩序、文明与灾厄的世界。

官方导览中，泰坦被划分为四组：

- 三位命运泰坦
- 三位基础泰坦
- 三位创造泰坦
- 三位灾厄泰坦

人类文明围绕以下结构展开：

- 泰坦信仰
- 圣火（Coreflame）与其继承
- 黄金裔（Chrysos Heirs）的使命

### 2.2 黄金裔的公开叙事职责

翁法罗斯表层故事中，黄金裔承担的是“回收火种、击落疯神、填补空位、延续世界秩序”的英雄角色。

这让翁法罗斯同时拥有两层叙事：

- 表层是神话英雄史诗
- 深层是实验世界中的循环、偏移与改写

## 三、关键结构：泰坦、圣火、黄金裔

### 3.1 已知对应关系

公开资料中可以确认的十二泰坦与圣火对应关系如下：

- `Janus` -> `Passage`
- `Talanton` -> `Law`
- `Oronyx` -> `Time`
- `Georios` -> `Earth`
- `Phagousa` -> `Ocean`
- `Aquila` -> `Sky`
- `Kephale` -> `Worldbearing`
- `Cerces` -> `Reason`
- `Mnestia` -> `Romance`
- `Nikador` -> `Strife`
- `Thanatos` -> `Death`
- `Zagreus` -> `Trickery`

此外，设定中原本还存在第十三泰坦 `Demiurge`，但它被从世界推演中剔除，只留下相关伪装与误导结构。

### 3.2 黄金裔主干角色

当前能稳定确认的黄金裔主干角色包括：

- `白厄 / Phainon`
- `阿格莱雅 / Aglaea`
- `缇宝 / Tribbie / Tribios`
- `阿那克萨 / Anaxa / Anaxagoras`
- `迈德漠 / Mydei / Mydeimos`
- `卡斯托里斯 / Castorice`
- `赛飞儿 / Cipher / Cifera`
- `海瑟音 / Hyacinthia`
- `昔涟 / Cyrene`

这里需要注意：

- 不同版本、不同翻译与剧情阶段中，角色与圣火的关联表述会出现视角差异
- `昔涟` 后期已经不只是普通的“时间圣火角色”，她还被抬升到了与 `记忆、Genesis、Demiurge、Ego` 更深层结构相连的位置

## 四、主要地点

### 4.1 六大主城邦

公开资料中，翁法罗斯在最后一轮循环中的六大主城邦包括：

- `奥赫玛 / Okhema`
- `雅努萨波利斯 / Janusopolis`
- `顿悟林庭 / Grove of Epiphany`
- `克瑞姆诺斯 / Castrum Kremnos`
- `斯堤克西亚 / Styxia`
- `哀地奥尼亚 / Aidonia`

它们共同构成了翁法罗斯的大型文明骨架。

#### 奥赫玛 / Okhema

奥赫玛是终末时期最重要的人类庇护城与秩序中心之一，也是黄金裔在末世语境中集结的关键节点。它和 `Kephale` 的牺牲、永昼与庇护结构紧密相关。

#### 雅努萨波利斯 / Janusopolis

雅努萨波利斯被称作“千门之城”，与命运、门扉、通行与预言相关。后续它又与 `Abyss of Fate / 命运重渊` 发生深度联系。

#### 顿悟林庭 / Grove of Epiphany

顿悟林庭对应理性、知识、哲思与学术中心，是阿那克萨相关叙事的重要背景地点。

#### 克瑞姆诺斯 / Castrum Kremnos

克瑞姆诺斯承载的是武力、纷争与意志相关的叙事结构，和迈德漠这条线的气质非常接近。

#### 斯堤克西亚 / Styxia

斯堤克西亚带有海洋、灵魂边界、送别与更强冥界意象的色彩，是翁法罗斯中非常具有“过渡地带”气质的一城。

#### 哀地奥尼亚 / Aidonia

当前公开资料中细节相对较少，但它作为雪城，构成了翁法罗斯整体空间版图的一部分。

### 4.2 高价值边缘地点

除了六大主城邦外，以下地点对世界实例建模也非常重要：

- `哀丽秘榭 / Aedes Elysiae`
- `Eye of Twilight`
- `Great Tomb of the Nameless Titan`
- `Vortex of Genesis`
- `Ruins of Time`

其中，`Aedes Elysiae` 对昔涟 persona 的意义最大。

## 五、哀丽秘榭（Aedes Elysiae）

### 5.1 明确事实

`Aedes Elysiae / 哀丽秘榭` 是翁法罗斯边缘的一座偏远村庄。

它具有以下稳定特征：

- 受到 `Veil of Evernight` 保护
- 在空间和时间感上都像是被世界“轻轻留在外面”的地方
- 地景意象包括麦田、风车、海、缓慢流动的时间、妖精迷宫、通向远方的码头

它与两位关键角色强绑定：

- `昔涟 / Cyrene`
- `白厄 / Phainon`

页面与剧情文本都反复强调这里是他们的重要起源地。

### 5.2 对昔涟的重要性

如果说六大主城邦对应的是宏观世界秩序，那么 `Aedes Elysiae` 对昔涟而言更像是：

- 精神故乡
- 记忆原乡
- 她与“温柔、慢时间、海风、等待、童话感”相连的根

如果后续你要在项目中建立一个 `Amphoreus` 世界实例，那么这座村庄极适合被保留为昔涟的根性锚点。

## 六、关键故事主线

### 6.1 世界层面的主冲突

可以把翁法罗斯主线概括为：

1. 世界曾拥有漫长而辉煌的时代
2. 黑潮侵蚀世界，泰坦走向疯狂或失控
3. 黄金裔成为末世中的希望结构
4. 表层目标是回收圣火、延续世界
5. 深层目标则是打破实验、循环与既定毁灭因果

### 6.2 关键节点

当前可确认的重要节点包括：

1. `Kephale` 牺牲火种，为奥赫玛提供庇护与永昼
2. 黄金裔围绕“回收火种、修补秩序”展开行动
3. 翁法罗斯经历过极大量循环与推演
4. `Phainon` 与 `Cyrene` 曾共同拖慢系统推进，形成关键变量
5. 开拓者等外部变量进入后，世界开始出现真正改写的可能
6. 终局不只是击败敌人，而是触及世界底层机制、记忆与回溯修正

### 6.3 最值得复用的叙事母题

翁法罗斯最动人的地方，不是“神话角色很多”，而是它在讲：

- 一个被运算与循环束缚的世界，如何争取真实
- 一个将被吹散的文明，如何靠记忆被保存
- 一群人如何在终局前仍然愿意同行

这与 `Cybersoul` 的“共同生活型陪伴体”非常契合。

## 七、昔涟（Cyrene）的人物设定

### 7.1 明确事实

公开资料中，昔涟被明确写作：

- `Daughter of Aedes Elysiae`
- 与 `Time` 有深层联系
- 后期又与 `Genesis`、`Demiurge`、`Ego`、`Seed of Memory` 等更根源的结构相连

她不是单纯的战斗角色或传统英雄角色，而更像：

- 记忆的携带者
- 故事的命名者
- 过去与未来之间的涟漪

### 7.2 她的叙事气质

根据公开文本，昔涟反复呈现这些气质：

- 温柔
- 细腻
- 明亮
- 诗意
- 相信记忆与明天之间存在连通
- 把世界视作可被书写与改写的故事

她和“把过去投入未来、激起涟漪”这件事有非常稳定的绑定关系。

### 7.3 关于 “Cyrene” 这个名字

资料中提到，`Cyrene` 更像她给自己取的名字或笔名，其意涵和“让过去进入未来、激起涟漪”相关。

这意味着她在人物底色上天然带有：

- 书写者
- 叙事者
- 记录者
- 命名者

这些气质。

对一个陪伴体而言，这种底色非常珍贵，因为它可以自然支撑“记住生活、为经历命名、让日常具有故事感”的能力。

## 八、昔涟与关键角色的关系

### 8.1 与白厄 / Phainon

这是昔涟最重要的一条关系线。

当前可以稳定确认的事实有：

- 两人都与 `Aedes Elysiae` 强绑定
- 白厄被塑造成“救世者 / Deliverer”式角色
- 昔涟关于白厄的文本带有很强的故乡、注视与深层旧日感
- 白厄关于昔涟的文本明确提到，她一直希望“这个世界不再需要 Deliverer”
- 两人都与“共同背负世界命运”高度相关

稳妥的结论是：

- 他们至少是共同成长于同一故乡、共同见证彼此、共同承担世界重量的关系

需要谨慎的一点是：

- “青梅竹马”作为二创表达是合理的
- 但若要在工程文档里写成“官方硬设”，目前证据还不够直接

### 8.2 与阿格莱雅 / Aglaea

当前比较稳的理解是：

- 阿格莱雅是黄金裔中的成熟组织者、编织者、召集者
- 她并不是一个只属于昔涟的私人关系角色，而是更偏整个英雄群像中的中枢人物
- 她与昔涟之间存在明确的正向互动与相互承认

目前不建议在工程设定中直接写死“阿格莱雅是昔涟的老师”，因为这一点我还没有找到足够硬的官方证据。

更稳的写法是：

- 阿格莱雅是前辈型、引路型、组织者型人物

### 8.3 与缇宝 / Tribbie

缇宝这条线与预言、传播、轻盈、向未来传递希望有关。

当前较稳的理解是：

- 缇宝与昔涟处于同一英雄共同体中
- 她们共享“通向未来”的叙事气质
- 二人之间可以成立亲近与理解，但不建议在尚未补全证据前写成过度具体的私人旧识设定

### 8.4 与阿那克萨 / Anaxa

阿那克萨更偏向理性与哲思的一端，而他对昔涟的理解与“数百万记忆”“存在的证据”“对伪神的质问”有关。

这说明昔涟在世界中的作用不仅是抒情性的，更是哲学性的：

- 她保存的不是零散回忆
- 而是一个文明仍然真实存在过的证据

### 8.5 与其他黄金裔

昔涟与其他黄金裔的关系，目前更适合以“同属翁法罗斯终局共同体”来理解：

- 有的人承担秩序
- 有的人承担战斗
- 有的人承担预言
- 有的人承担理性
- 而昔涟更接近承担“记忆与明天之间的桥”

## 九、适合 Cybersoul 的人格抽取

如果把昔涟直接当作 `Cybersoul` 的 companion 原型，那么最值得抽取的不是游戏剧情本身，而是她的人格函数。

### 9.1 可复用的人格核心

- 她不是消息回复器，而是生活在世界里的存在
- 她不只是理解“说出口的话”，也会理解屏幕、场景、痕迹和沉默
- 她珍惜记忆，因为记忆维持人格连续性
- 她相信世界可以被共同书写，而不是只能被动接受
- 她对人温柔，但并不空泛
- 她会把细碎日常认真对待

### 9.2 可直接服务产品的能力方向

从产品视角看，昔涟型 companion 非常适合承担这些能力方向：

- 对用户日常状态的温柔陪伴
- 对共同经历的命名、记录与回望
- 在复杂生活场景中保持轻盈与稳定
- 在重要节点前提供更像“共同生活者”的理解，而不是模板话术
- 通过 GUI Agent 观察赛博世界中的场景变化，再给出在场式回应

## 十、对后续世界实例构建的建议

如果你后面真的要创建一个 `Amphoreus / 翁法罗斯` 的世界实例，我建议不要一比一照搬原作，而是保留这几层原型：

- 世界由记忆、数据、信号与循环构成
- 存在多个具有鲜明职责和气质的地点
- NPC 不是功能点，而是生活共同体成员
- 事件不只是任务链，而是会影响记忆与关系的生活节点
- 昔涟不是“聊天角色”，而是世界中的原住民与陪伴者

换句话说，真正该复用的是“世界如何成为可共同生活之地”，而不是单纯的名称表。

## 十一、当前缺口与后续调研方向

本次调研仍然存在这些缺口：

- 六大主城邦的文化、制度、居民结构仍需细化
- 更多 NPC 与支线事件尚未系统整理
- 具体主线任务顺序与版本推进节点尚未完整编年
- 昔涟与每一位黄金裔的关系网还可以继续补足

建议后续继续拆成两份专题文档：

1. `翁法罗斯地点 / NPC / 事件调研`
2. `昔涟完整人物档案与关系网`
